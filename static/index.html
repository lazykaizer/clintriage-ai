What does the AI Agent do?
📋

1. Receives Incoming Patients

The ER environment sends a batch of incoming patients with their vitals, symptoms, and arrival details to the agent.

🔍

2. Analyzes Clinical Urgency

The agent evaluates each patient's heart rate, SpO₂, blood pressure, chief complaint, and arrival mode to determine urgency.

📊

3. Builds a Priority Queue

Based on the analysis, the agent ranks all patients from most urgent to least urgent — creating a triage priority queue.

✅

4. Graded by the Environment

The ranking is submitted to the grader, which compares it against the clinically correct order and returns a reward score.

Agent Scope: The agent only lists and prioritizes incoming patients. It does not manage hospital records, enter patient details, or handle internal operations.
